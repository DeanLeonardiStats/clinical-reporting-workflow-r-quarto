1 Abstract

A longitudinal analysis for Parkinson’s patient UPDRS scores using MAR multiple imputation (MICE) sensitivity analysis and propensity score matching: a non-randomized, observational, simulated data study

2 Executive Summary

This report demonstrates a reproducible longitudinal clinical reporting workflow using R and Quarto in a simulated Parkinson’s disease observational study. Analyses include:

  • longitudinal mixed-effects modeling,
  • MAR multiple imputation sensitivity analysis (MICE),
  • propensity score matching,
  • treatment effect estimation,
  • and automated HTML reporting.

This workflow is intended as a methodological demonstration and does not represent a regulatory submission analysis.

3 Background

We assessed the efficacy of treatment in patients with Parkinson’s disease (PD) using UPDRS (Unified Parkinson’s Disease Rating Scale) scores. In PD research, the UPDRS is the gold standard for measuring motor and non-motor symptoms.

4 Methods

This was a non-randomized, observational, 2-arm, simulated data study for adults with ages sampled from a normal distribution with mean 70 and standard deviation 12 years. Sex was sampled from a Bernoulli distribution with probability of being female 0.6. Treatment groups were assigned based on age and sex (non-random, simulating selection bias). We simulated longitudinal data for Parkinson’s patients with UPDRS scores measured at baseline, 1 year, and 2 years. The two groups were control and treatment. Longitudinal analysis was used to track changes in UPDRS scores across multiple time points to determine if treatment slows or accelerates disease progression and identifies symptom trajectories (e.g., motor decline, non-motor worsening) and supports trial design and stratification. The study data was simulated so that older and more females tended to get treatment. The primary endpoint was the differences between treatment groups (treatment - control) in UPDRS scores at year 2. No safety analysis was conducted. The ITT analysis set included all patients in the simulated data set. Multiple imputation by chained equations (MICE) under a missing-at-random (MAR) assumption, with M = 25 completed datasets pooled via Rubin’s Rules, and propensity score matching to balance age and sex were utilized for sensitivity analyses.

set.seed(456)
# Non-randomized observational study: older and more females tend to get treatment
n_patients <- 400
patients <- 1:n_patients
times <- c(0, 1, 2)  # baseline, 1 year, 2 years

# Non-random group assignment: function for treatment probability
prob_treatment <- function(age, sex) {
  plogis(-1 + 0.05*(age - 65) + 0.6*(sex == "F"))
}

# Create baseline covariates first
baseline_covariates <- data.frame(
  Patient = as.factor(patients),
  Age = rnorm(n_patients, mean = 70, sd = 12),  # Older population
  Sex = sample(c("M", "F"), n_patients, replace = TRUE, prob = c(0.4, 0.6))  # More females
)

# Assign groups based on age and sex (non-random, simulating selection bias)
baseline_covariates$Group <- as.factor(sapply(1:n_patients, function(i) {
  if (runif(1) < prob_treatment(baseline_covariates$Age[i], baseline_covariates$Sex[i])) {
    "Treatment"
  } else {
    "Control"
  }
}))

# Create longitudinal data frame
data <- expand.grid(Patient = patients, Time = times)
data$Patient <- as.factor(data$Patient)
data$Time <- as.factor(data$Time)

# Merge with baseline covariates
data <- data %>% left_join(baseline_covariates, by = "Patient")

# Patient-level random intercepts — generates realistic between-patient variance
# SD_u = 5 (between-patient), SD_eps = 5 (within-patient) → ICC ≈ 0.50
# Without this, lme4 estimates σ²_u ≈ 0 and the mixed model offers no advantage
# over standard regression, undermining the methodological rationale.
patient_re <- rnorm(n_patients, mean = 0, sd = 5)

# Simulate UPDRS scores with patient random intercepts
data$UPDRS <- rnorm(nrow(data),
  mean = 35 + 5 * as.numeric(as.character(data$Time)) +
         ifelse(data$Group == "Treatment", -1.5, 0) +
         patient_re[as.numeric(data$Patient)],
  sd = 5)

# Parameterization of missing data
data$UPDRS <- ifelse(
  runif(nrow(data)) < params$missing_rate &
  data$Time != "0",
  NA,
  data$UPDRS
)

# ── Intraclass Correlation Coefficient (ICC) ──────────────────────────────────
# ICC = σ²_u / (σ²_u + σ²): proportion of total variance attributable to
# between-patient differences. A high ICC (> 0.5) confirms that patients differ
# substantially at baseline and validates the need for a mixed model — standard
# regression would underestimate standard errors by treating within-person
# observations as independent.

vc       <- as.data.frame(VarCorr(m1))
var_u    <- vc$vcov[vc$grp == "Patient"]   # between-patient variance
var_eps  <- vc$vcov[vc$grp == "Residual"]  # within-patient (residual) variance
icc_val  <- var_u / (var_u + var_eps)

6 Findings

Four hundred patients were assigned to receive control (n=234), or Treatment (n=166). Treatment reduced UPDRS score versus control (treatment 43.35 vs control 46.27; difference -2.26, 95% CI (-3.797, -0.718)).

7 Interpretation

In patients with Parkinson’s disease, treatment reduced the UPDRS score at Year 2. These findings are based on simulated data and are intended as a methodological demonstration only.

7.1 Missing Data Analysis

Under a Missing At Random (MAR) assumption, missing UPDRS outcomes are predicted by observed covariates (Age, Sex, Group) and observed UPDRS values at other timepoints. Multiple imputation by chained equations (MICE) with predictive mean matching was used to generate M = 25 completed datasets. Fixed-effect estimates from each dataset were combined using Rubin’s Rules to produce a single pooled estimate and standard error that properly accounts for within- and between-imputation uncertainty.

pred <- ini$predictorMatrix

# Baseline UPDRS is fully observed — exclude from imputation target
# but retain as a predictor for Year 1 and Year 2 imputation
meth["UPDRS_t0"] <- ""
pred["UPDRS_t0", ] <- 0

# pmm (predictive mean matching) is appropriate for bounded continuous scores:

# empirical distribution and avoiding impossible out-of-range imputations

  m               = 25,    # 25 datasets: adequate for ~25% missingness
  method          = meth,
  predictorMatrix = pred,
  maxit           = 15,    # iterations per dataset until convergence
  print           = FALSE
)

# For each of the M imputed wide datasets:
#   (a) re-attach Patient IDs
#   (b) reshape wide → long
#   (c) fit the same LMER specification as the primary analysis

fit_on_completed <- function(i) {

  wide_i <- mice::complete(imp_mar, action = i) |>
    mutate(Patient = patient_ids)                  # restore patient ID

  long_i <- wide_i |>
    pivot_longer(
      cols         = starts_with("UPDRS_t"),

# ── 7. Pooled treatment effect at Year 2 formatted like primary analysis ──────
# emmeans is computed for each imputed model, then estimates and SEs are
# combined using Rubin's Rules. The Satterthwaite df from emmeans is used
# in the Barnard-Rubin (1999) degrees-of-freedom approximation.

pool_emmeans_contrast <- function(models, time_level = "2") {

  # Extract Treatment - Control contrast at the requested timepoint
  extract_one <- function(mod) {
    em  <- emmeans(mod, ~ Group | Time)
    ct  <- contrast(em, method = "trt.vs.ctrl", ref = "Control")
    row <- as.data.frame(ct) |> filter(Time == time_level)
    list(
      est = row$estimate,
      se  = row$SE,
      df  = row$df      # Satterthwaite df
    )
  }

  res   <- lapply(models, extract_one)
  M     <- length(res)
  ests  <- sapply(res, `[[`, "est")
  ses   <- sapply(res, `[[`, "se")
  dfs   <- sapply(res, `[[`, "df")

  # Rubin's Rules
  Q_bar <- mean(ests)                          # pooled estimate
  U_bar <- mean(ses^2)                         # average within-imputation variance
  B     <- var(ests)                           # between-imputation variance
  T_var <- U_bar + (1 + 1 / M) * B            # total variance
  SE_p  <- sqrt(T_var)

  # Fraction of missing information (FMI)
  r   <- (1 + 1 / M) * B / U_bar
  fmi <- r / (1 + r)

  # Barnard-Rubin (1999) pooled degrees of freedom
  df_old <- mean(dfs)
  df_m   <- (M - 1) / r^2
  df_obs <- (df_old + 1) / (df_old + 3) * df_old * (1 - r)
  df_p   <- 1 / (1 / df_m + 1 / df_obs)

  ci <- Q_bar + qt(c(0.025, 0.975), df = df_p) * SE_p

  tibble(
    Contrast    = "Treatment − Control",
    Time        = paste0("Year ", time_level),
    Estimate    = round(Q_bar, 3),
    SE          = round(SE_p,  3),
    CI          = paste0("(", round(ci[1], 2), ", ", round(ci[2], 2), ")"),
    FMI         = round(fmi, 3)
  )
}

contrast_pooled <- pool_emmeans_contrast(models_mar, time_level = "2")

## Background

We assessed the efficacy of treatment in patients with Parkinson's disease (PD) using UPDRS (Unified Parkinson's Disease Rating Scale) scores. In PD research, the UPDRS is the gold standard for measuring motor and non-motor symptoms.

## Methods

This was a non-randomized, observational, 2-arm, simulated data study for adults with ages sampled from a normal distribution with mean 70 and standard deviation 12 years. Sex was sampled from a Bernoulli distribution with probability of being female 0.6. Treatment groups were assigned based on age and sex (non-random, simulating selection bias). We simulated longitudinal data for Parkinson's patients with UPDRS scores measured at baseline, 1 year, and 2 years. The two groups were control and treatment. Longitudinal analysis was used to track changes in UPDRS scores across multiple time points to determine if treatment slows or accelerates disease progression and identifies symptom trajectories (e.g., motor decline, non-motor worsening) and supports trial design and stratification. The study data was simulated so that older and more females tended to get treatment. The primary endpoint was the differences between treatment groups (treatment - control) in UPDRS scores at year 2. No safety analysis was conducted. The ITT analysis set included all patients in the simulated data set. Multiple imputation by chained equations (MICE) under a missing-at-random (MAR) assumption, with M = 25 completed datasets pooled via Rubin's Rules, and propensity score matching to balance age and sex were utilized for sensitivity analyses.

#| label: simulate-data
set.seed(456)
# Non-randomized observational study: older and more females tend to get treatment
n_patients <- 400
patients <- 1:n_patients
times <- c(0, 1, 2)  # baseline, 1 year, 2 years

# Non-random group assignment: function for treatment probability
prob_treatment <- function(age, sex) {
  plogis(-1 + 0.05*(age - 65) + 0.6*(sex == "F"))
}

# Create baseline covariates first
baseline_covariates <- data.frame(
  Patient = as.factor(patients),
  Age = rnorm(n_patients, mean = 70, sd = 12),  # Older population
  Sex = sample(c("M", "F"), n_patients, replace = TRUE, prob = c(0.4, 0.6))  # More females
)

# Assign groups based on age and sex (non-random, simulating selection bias)
baseline_covariates$Group <- as.factor(sapply(1:n_patients, function(i) {
  if (runif(1) < prob_treatment(baseline_covariates$Age[i], baseline_covariates$Sex[i])) {
    "Treatment"
  } else {
    "Control"
  }
}))

# Create longitudinal data frame
data <- expand.grid(Patient = patients, Time = times)
data$Patient <- as.factor(data$Patient)
data$Time <- as.factor(data$Time)

# Merge with baseline covariates
data <- data %>% left_join(baseline_covariates, by = "Patient")

# Patient-level random intercepts — generates realistic between-patient variance
# SD_u = 5 (between-patient), SD_eps = 5 (within-patient) → ICC ≈ 0.50
# Without this, lme4 estimates σ²_u ≈ 0 and the mixed model offers no advantage
# over standard regression, undermining the methodological rationale.
patient_re <- rnorm(n_patients, mean = 0, sd = 5)

# Simulate UPDRS scores with patient random intercepts
data$UPDRS <- rnorm(nrow(data),
  mean = 35 + 5 * as.numeric(as.character(data$Time)) +
         ifelse(data$Group == "Treatment", -1.5, 0) +
         patient_re[as.numeric(data$Patient)],
  sd = 5)

# Parameterization of missing data
data$UPDRS <- ifelse(
  runif(nrow(data)) < params$missing_rate &
  data$Time != "0",
  NA,
  data$UPDRS
)

#| label: icc

# ── Intraclass Correlation Coefficient (ICC) ──────────────────────────────────
# ICC = σ²_u / (σ²_u + σ²): proportion of total variance attributable to
# between-patient differences. A high ICC (> 0.5) confirms that patients differ
# substantially at baseline and validates the need for a mixed model — standard
# regression would underestimate standard errors by treating within-person
# observations as independent.

vc       <- as.data.frame(VarCorr(m1))
var_u    <- vc$vcov[vc$grp == "Patient"]   # between-patient variance
var_eps  <- vc$vcov[vc$grp == "Residual"]  # within-patient (residual) variance
icc_val  <- var_u / (var_u + var_eps)

tibble(
  Component              = c("Between-patient (σ²_u)", "Within-patient / Residual (σ²)", "ICC"),
  Variance               = c(round(var_u, 3), round(var_eps, 3), NA_real_),
  Value                  = c(NA_character_, NA_character_, round(icc_val, 3))

### Missing Data Analysis

Under a Missing At Random (MAR) assumption, missing UPDRS outcomes are predicted by observed covariates (Age, Sex, Group) and observed UPDRS values at other timepoints. Multiple imputation by chained equations (MICE) with predictive mean matching was used to generate M = 25 completed datasets. Fixed-effect estimates from each dataset were combined using Rubin's Rules to produce a single pooled estimate and standard error that properly accounts for within- and between-imputation uncertainty.

# but retain as a predictor for Year 1 and Year 2 imputation
meth["UPDRS_t0"] <- ""
pred["UPDRS_t0", ] <- 0

# pmm (predictive mean matching) is appropriate for bounded continuous scores:
# it draws observed values close to the linear prediction, preserving the
# empirical distribution and avoiding impossible out-of-range imputations
meth[c("UPDRS_t1", "UPDRS_t2")] <- "pmm"

# ── 3. Run mice ────────────────────────────────────────────────────────────────
set.seed(2026)
imp_mar <- mice(
  data_mice,
  m               = 25,    # 25 datasets: adequate for ~25% missingness
  method          = meth,
  predictorMatrix = pred,
  maxit           = 15,    # iterations per dataset until convergence
  print           = FALSE
)

```{r}
#| label: mice-emmeans-pool
#| message: false
#| warning: false

# ── 7. Pooled treatment effect at Year 2 formatted like primary analysis ──────
# emmeans is computed for each imputed model, then estimates and SEs are
# combined using Rubin's Rules. The Satterthwaite df from emmeans is used
# in the Barnard-Rubin (1999) degrees-of-freedom approximation.

pool_emmeans_contrast <- function(models, time_level = "2") {

  # Extract Treatment - Control contrast at the requested timepoint
  extract_one <- function(mod) {
    em  <- emmeans(mod, ~ Group | Time)
    ct  <- contrast(em, method = "trt.vs.ctrl", ref = "Control")
    row <- as.data.frame(ct) |> filter(Time == time_level)
    list(
      est = row$estimate,
      se  = row$SE,
      df  = row$df      # Satterthwaite df
    )
  }

  res   <- lapply(models, extract_one)
  M     <- length(res)
  ests  <- sapply(res, `[[`, "est")
  ses   <- sapply(res, `[[`, "se")
  dfs   <- sapply(res, `[[`, "df")

  # Rubin's Rules
  Q_bar <- mean(ests)                          # pooled estimate
  U_bar <- mean(ses^2)                         # average within-imputation variance
  B     <- var(ests)                           # between-imputation variance
  T_var <- U_bar + (1 + 1 / M) * B            # total variance
  SE_p  <- sqrt(T_var)

  # Fraction of missing information (FMI)
  r   <- (1 + 1 / M) * B / U_bar
  fmi <- r / (1 + r)

  # Barnard-Rubin (1999) pooled degrees of freedom
  df_old <- mean(dfs)
  df_m   <- (M - 1) / r^2
  df_obs <- (df_old + 1) / (df_old + 3) * df_old * (1 - r)
  df_p   <- 1 / (1 / df_m + 1 / df_obs)

  ci <- Q_bar + qt(c(0.025, 0.975), df = df_p) * SE_p

  tibble(
    Contrast    = "Treatment − Control",
    Time        = paste0("Year ", time_level),
    Estimate    = round(Q_bar, 3),
    SE          = round(SE_p,  3),
    CI          = paste0("(", round(ci[1], 2), ", ", round(ci[2], 2), ")"),
    FMI         = round(fmi, 3)
  )
}

contrast_pooled <- pool_emmeans_contrast(models_mar, time_level = "2")

contrast_pooled |>
  gt() |>
  tab_header(
    title    = "Table 3c. Pooled Treatment Effect at Year 2 — MAR Sensitivity Analysis",
    subtitle = "MICE / Predictive Mean Matching; m = 25; Rubin's Rules"
  ) |>
  cols_label(
    Contrast = "Contrast",
    Time     = "Timepoint",
    Estimate = "Estimate",
    SE       = "Std. Error",
    CI       = "95% CI",
    FMI      = "FMI"
  ) |>
  tab_style(
    style     = cell_text(weight = "bold"),
    locations = cells_column_labels()
  ) |>
  tab_footnote(
    footnote = paste0(
      "FMI: Fraction of Missing Information — proportion of total variance ",
      "attributable to missing data. Higher values indicate stronger sensitivity ",
      "to the MAR assumption. df computed via Barnard-Rubin (1999) approximation."
    )
  ) |>
  tab_options(
    table.font.size           = px(13),
    heading.title.font.size   = px(14),
    heading.title.font.weight = "bold",
    table.width               = pct(100)
  )

The Fraction of Missing Information (FMI) quantifies the proportion of uncertainty attributable to missing data; values below 0.10 are generally considered low.

This report demonstrates a reproducible and automated longitudinal clinical reporting workflow using modern R and Quarto methodologies. ## Limitations

This report uses simulated observational data for methodological demonstration purposes. Results should not be interpreted clinically. Missing data sensitivity analyses use MICE with predictive mean matching under a missing-at-random (MAR) assumption. Whether the MAR assumption holds cannot be verified from the observed data alone; MNAR sensitivity analyses (e.g., reference-based methods using the rbmi package) would be required for regulatory-grade submissions.

## Reproducibility

This report was generated using a reproducible Quarto workflow in R.\
All analyses, figures, tables, and statistical outputs were generated programmatically from simulated source data.

Key components include:

- automated data simulation,
- longitudinal mixed-effects modeling,
- MAR multiple imputation using MICE (m = 25, Rubin's Rules),
- propensity score matching,
- reproducible statistical reporting,
- and parameterized report generation.

The workflow is designed to reflect modern clinical reporting and real-world evidence analysis practices used in pharmaceutical and biotechnology settings.
