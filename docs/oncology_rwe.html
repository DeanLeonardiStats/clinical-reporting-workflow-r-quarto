This vignette demonstrates a reproducible RWE survival analysis pipeline applied to a simulated oncology RCT (n = 400) with overall survival (OS) as the primary endpoint. Methods progress from descriptive Kaplan–Meier estimation through multivariable Cox regression, inverse probability weighting (IPW) for the average treatment effect (ATE), and E-value sensitivity analysis for unmeasured confounding — mirroring regulatory RWE practice for oncology submissions.

3 IPW / AIPW for ATE with Robust SE

Doubly robust estimation on oncology data.

  • WeightIt: propensity score model (trt ~ age + stage + ecog), estimand=“ATE”
  • IPW-weighted Cox model for ATE with sandwich SE (regulatory practice)
  • E-value for the primary IPW HR (unmeasured confounding sensitivity)
Show code 
# IPTW model
w_out   <- weightit(trt ~ age + stage + ecog, data = data_onc,
                    method = "glm", estimand = "ATE")
cox_ipw <- coxph(Surv(os_time, os_event) ~ trt,
                 data = data_onc, weights = w_out$weights, robust = TRUE)

# Robust HR and CI
s    <- summary(cox_ipw)
beta <- coef(cox_ipw)["trt"]
se   <- s$coefficients["trt", "robust se"]
hr   <- exp(beta)
lo   <- exp(beta - qnorm(0.975) * se)
hi   <- exp(beta + qnorm(0.975) * se)

# E-values — evalues.HR() returns a matrix; extract numerically before building tibble
ev_mat <- evalues.HR(est = hr, lo = lo, hi = hi, rare = FALSE)
e_est  <- as.numeric(ev_mat[1, 1])
e_ci   <- as.numeric(ev_mat[1, 2])

results_tbl <- tibble(
  Covariate       = "trt",
  HR              = hr,
  Lower95CI       = lo,
  Upper95CI       = hi,
  EValue_Estimate = e_est,
  EValue_CI_Limit = e_ci
)

results_tbl |>
  gt() |>
  fmt_number(
    columns  = c(HR, Lower95CI, Upper95CI, EValue_Estimate, EValue_CI_Limit),
    decimals = 2
  ) |>
  cols_label(
    Covariate       = "Treatment",
    HR              = "Hazard Ratio",
    Lower95CI       = "95% CI Lower",
    Upper95CI       = "95% CI Upper",
    EValue_Estimate = "E-value",
    EValue_CI_Limit = "E-value (CI Limit)"
  ) |>
  tab_header(title = "IPW-Weighted Cox Model with E-value Sensitivity Analysis") |>
  tab_footnote(footnote = "ATE IPTW model using age, stage, and ECOG score.")

4 Doubly Robust AIPW for RMST Difference

Augmented IPW (AIPW) combines an outcome model with the propensity score to achieve double robustness — consistent even if one model is misspecified.

Show code 
# Outcome models: E[RMST24 | X, A=1] and E[RMST24 | X, A=0]
data_onc$rmst24_tmp <- pmin(data_onc$os_time, 24)
mu1 <- predict(lm(rmst24_tmp ~ age + stage + ecog,
                  data = data_onc[data_onc$trt == 1, ]),
               newdata = data_onc)
mu0 <- predict(lm(rmst24_tmp ~ age + stage + ecog,
                  data = data_onc[data_onc$trt == 0, ]),
               newdata = data_onc)

# Propensity scores from WeightIt object
ps  <- w_out$ps
A   <- data_onc$trt
Y   <- data_onc$rmst24_tmp

# AIPW influence function
phi <- (A / ps) * (Y - mu1) + mu1 -
       ((1 - A) / (1 - ps)) * (Y - mu0) - mu0

aipw_est <- mean(phi)
aipw_se  <- sd(phi) / sqrt(length(phi))
aipw_lo  <- aipw_est - qnorm(0.975) * aipw_se
aipw_hi  <- aipw_est + qnorm(0.975) * aipw_se
aipw_z   <- aipw_est / aipw_se
aipw_p   <- 2 * pnorm(-abs(aipw_z))

tibble(
  Analysis  = "AIPW — RMST Difference at 24 Months",
  Estimate  = aipw_est,
  SE        = aipw_se,
  `95% CI Lower` = aipw_lo,
  `95% CI Upper` = aipw_hi,
  `P-value` = aipw_p
) |>
  gt() |>
  fmt_number(columns = c(Estimate, SE, `95% CI Lower`, `95% CI Upper`), decimals = 3) |>
  fmt_scientific(columns = `P-value`, decimals = 2) |>
  tab_header(title = "Doubly Robust AIPW Estimator") |>
  tab_footnote(
    footnote = "Augmented IPW using outcome models fit separately by treatment arm (OLS on RMST24) and propensity scores from WeightIt. Doubly robust: consistent if either the outcome model or propensity score model is correctly specified."
  ) |>
  tab_options(
    table.font.size           = px(13),
    heading.title.font.size   = px(14),
    heading.title.font.weight = "bold",
    table.width               = pct(100)
  )

5 Causal Forest for Heterogeneous Treatment Effects

Show code 
# RMST at month 24 as continuous outcome
data_onc$rmst24 <- pmin(data_onc$os_time, 24)


# Covariate Matrix
X <- model.matrix(
  ~ age + stage + ecog,
  data = data_onc
)[,-1]


# Fit Causal Forest
W <- data_onc$trt
Y <- data_onc$rmst24

set.seed(42)

cf <- causal_forest(
  X,
  Y,
  W,
  num.trees = 2000,
  honesty = TRUE
)



# Individual Treatment Effects (CATEs)
tau_hat <- predict(cf)$predictions
data_onc$tau_hat <- tau_hat

Interpretation: - Positive τ̂ → treatment increases RMST - Negative τ̂ → treatment decreases RMST

8 Causal Forest Calibration Test

Show code 
cal <- test_calibration(cf)

# Extract directly from the matrix by position — avoids as.data.frame() column
# name instability across grf versions
cal_tbl <- tibble(
  Term     = rownames(cal),
  Estimate = cal[, 1],
  SE       = cal[, 2]

  mutate(
    CI_Lower = Estimate - 1.96 * SE,
    CI_Upper = Estimate + 1.96 * SE,

    decimals = 2
  ) |>
  cols_label(

    Estimate = "Estimate",

    CI_Upper = "95% CI Upper"
  ) |>

    title = "Causal Forest Calibration Test"

Table 5. Causal Forest Calibration Test.
Causal Forest Calibration Test
Calibration Component Estimate SE 95% CI Lower 95% CI Upper P-value
mean.forest.prediction 1.014 0.283 0.459 1.569 3.41 × 10−4
differential.forest.prediction 0.177 0.594 −0.987 1.342 7.65 × 10−1

The calibration test assesses whether the causal forest’s predicted CATEs (τ̂) are meaningful signals rather than noise. Two components are reported. The mean forest prediction tests whether the average predicted CATE differs from zero — a significant estimate (p < 0.05) confirms that the forest detects a non-zero average treatment effect, consistent with the IPW and AIPW estimates above. The differential forest prediction is the more diagnostic quantity: it tests whether variation in τ̂ across individuals predicts variation in actual outcomes after accounting for the mean. An estimate near 1.0 with a significant p-value indicates well-calibrated heterogeneity — the forest’s rankings of who benefits most are trustworthy. An estimate near 0 suggests the predicted heterogeneity is not supported by the data and individual CATEs should be interpreted with caution.

9 Causal Forest Variable Importance

Show code 
vi_tbl <-
  tibble(
    Variable = colnames(X),
    Importance = variable_importance(cf)
  ) |>
  arrange(desc(Importance)) |>
  mutate(
    Rank = row_number(),
    Relative_Importance =
      Importance / max(Importance)
  )

vi_tbl |>
  select(
    Rank,
    Variable,
    Importance,
    Relative_Importance
  ) |>
  gt() |>
  fmt_number(
    columns = c(Importance, Relative_Importance),
    decimals = 3
  ) |>
  cols_label(
    Rank = "Rank",
    Variable = "Covariate",
    Importance = "Variable Importance",
    Relative_Importance = "Relative Importance"
  ) |>
  tab_header(
    title = "Causal Forest Variable Importance"
  )
Table 6. Causal Forest Variable Importance.
Causal Forest Variable Importance
Rank Covariate Variable Importance Relative Importance
1 age 0.548 1.000
2 ecog 0.241 0.439
3 stage 0.211 0.385

Variable importance in a causal forest measures how frequently each covariate is used as a splitting variable across trees, weighted by the depth at which splits occur — earlier splits receive higher weight because they partition the largest portions of the data. The top-ranked covariate is the primary driver of treatment effect heterogeneity in this simulation: patients differ most in their RMST benefit along that dimension. Relative importance rescales each score to the maximum, making cross-covariate comparisons straightforward — a value of 1.0 is the most influential, values near 0 contribute minimally to heterogeneity. In a precision medicine context, the highest-importance covariate is the natural candidate for subgroup stratification: patients in the upper versus lower quantiles of that variable represent the population segment with the most divergent treatment responses. These importance scores should be interpreted as measures of predictive splitting activity, not as causal effect estimates for the covariates themselves.

## Doubly Robust AIPW for RMST Difference

Augmented IPW (AIPW) combines an outcome model with the propensity score to achieve double robustness — consistent even if one model is misspecified.

```{r}
#| label: aipw-dr
#| tbl-cap: "Table 2b. Doubly Robust AIPW Estimate — RMST Difference at 24 Months."
#| warning: false
#| message: false

# Outcome models: E[RMST24 | X, A=1] and E[RMST24 | X, A=0]
data_onc$rmst24_tmp <- pmin(data_onc$os_time, 24)
mu1 <- predict(lm(rmst24_tmp ~ age + stage + ecog,
                  data = data_onc[data_onc$trt == 1, ]),
               newdata = data_onc)
mu0 <- predict(lm(rmst24_tmp ~ age + stage + ecog,
                  data = data_onc[data_onc$trt == 0, ]),
               newdata = data_onc)

# Propensity scores from WeightIt object
ps  <- w_out$ps
A   <- data_onc$trt
Y   <- data_onc$rmst24_tmp

# AIPW influence function
phi <- (A / ps) * (Y - mu1) + mu1 -
       ((1 - A) / (1 - ps)) * (Y - mu0) - mu0

aipw_est <- mean(phi)
aipw_se  <- sd(phi) / sqrt(length(phi))
aipw_lo  <- aipw_est - qnorm(0.975) * aipw_se
aipw_hi  <- aipw_est + qnorm(0.975) * aipw_se
aipw_z   <- aipw_est / aipw_se
aipw_p   <- 2 * pnorm(-abs(aipw_z))

tibble(
  Analysis  = "AIPW — RMST Difference at 24 Months",
  Estimate  = aipw_est,
  SE        = aipw_se,
  `95% CI Lower` = aipw_lo,
  `95% CI Upper` = aipw_hi,
  `P-value` = aipw_p
) |>
  gt() |>
  fmt_number(columns = c(Estimate, SE, `95% CI Lower`, `95% CI Upper`), decimals = 3) |>
  fmt_scientific(columns = `P-value`, decimals = 2) |>
  tab_header(title = "Doubly Robust AIPW Estimator") |>
  tab_footnote(
    footnote = "Augmented IPW using outcome models fit separately by treatment arm (OLS on RMST24) and propensity scores from WeightIt. Doubly robust: consistent if either the outcome model or propensity score model is correctly specified."
  ) |>
  tab_options(
    table.font.size           = px(13),
    heading.title.font.size   = px(14),
    heading.title.font.weight = "bold",
    table.width               = pct(100)
  )
```

## Causal Forest for Heterogeneous Treatment Effects

```{r}
#| label: causal-forest
#| warning: false
#| message: false



# RMST at month 24 as continuous outcome
data_onc$rmst24 <- pmin(data_onc$os_time, 24)


# Covariate Matrix
X <- model.matrix(
  ~ age + stage + ecog,
  data = data_onc
)[,-1]


# Fit Causal Forest
W <- data_onc$trt
Y <- data_onc$rmst24

set.seed(42)

cf <- causal_forest(
  X,
  Y,
  W,
  num.trees = 2000,
  honesty = TRUE
)



# Individual Treatment Effects (CATEs)
tau_hat <- predict(cf)$predictions
data_onc$tau_hat <- tau_hat


```

Interpretation:
- Positive τ̂ → treatment increases RMST
- Negative τ̂ → treatment decreases RMST

## Causal Forest Calibration Test

```{r}
#| label: calibration-test
#| tbl-cap: "Table 5. Causal Forest Calibration Test."
#| warning: false
#| message: false

cal <- test_calibration(cf)

# Extract directly from the matrix by position — avoids as.data.frame() column
# name instability across grf versions
cal_tbl <- tibble(
  Term     = rownames(cal),
  Estimate = cal[, 1],
  SE       = cal[, 2]
) |>
  mutate(
    CI_Lower = Estimate - 1.96 * SE,
    CI_Upper = Estimate + 1.96 * SE,
    `P-value` = 2 * pnorm(-abs(Estimate / SE))
  )

cal_tbl |>
  gt() |>
  fmt_number(
    columns = c(Estimate, SE, CI_Lower, CI_Upper),
    decimals = 3
  ) |>
  fmt_scientific(
    columns = `P-value`,
    decimals = 2
  ) |>
  cols_label(
    Term = "Calibration Component",
    Estimate = "Estimate",
    SE = "SE",
    CI_Lower = "95% CI Lower",
    CI_Upper = "95% CI Upper"
  ) |>
  tab_header(
    title = "Causal Forest Calibration Test"
  )
```

The calibration test assesses whether the causal forest's predicted CATEs (τ̂) are meaningful signals rather than noise. Two components are reported. The **mean forest prediction** tests whether the average predicted CATE differs from zero — a significant estimate (p < 0.05) confirms that the forest detects a non-zero average treatment effect, consistent with the IPW and AIPW estimates above. The **differential forest prediction** is the more diagnostic quantity: it tests whether variation in τ̂ across individuals predicts variation in actual outcomes after accounting for the mean. An estimate near 1.0 with a significant p-value indicates well-calibrated heterogeneity — the forest's rankings of who benefits most are trustworthy. An estimate near 0 suggests the predicted heterogeneity is not supported by the data and individual CATEs should be interpreted with caution.

## Causal Forest Variable Importance

```{r}
#| label: variable-importance
#| tbl-cap: "Table 6. Causal Forest Variable Importance."
#| warning: false
#| message: false

vi_tbl <-
  tibble(
    Variable = colnames(X),
    Importance = variable_importance(cf)
  ) |>
  arrange(desc(Importance)) |>
  mutate(
    Rank = row_number(),
    Relative_Importance =
      Importance / max(Importance)
  )

vi_tbl |>
  select(
    Rank,
    Variable,
    Importance,
    Relative_Importance
  ) |>
  gt() |>
  fmt_number(
    columns = c(Importance, Relative_Importance),
    decimals = 3
  ) |>
  cols_label(
    Rank = "Rank",
    Variable = "Covariate",
    Importance = "Variable Importance",
    Relative_Importance = "Relative Importance"
  ) |>
  tab_header(
    title = "Causal Forest Variable Importance"
  )

```

Variable importance in a causal forest measures how frequently each covariate is used as a splitting variable across trees, weighted by the depth at which splits occur — earlier splits receive higher weight because they partition the largest portions of the data. The **top-ranked covariate** is the primary driver of treatment effect heterogeneity in this simulation: patients differ most in their RMST benefit along that dimension. **Relative importance** rescales each score to the maximum, making cross-covariate comparisons straightforward — a value of 1.0 is the most influential, values near 0 contribute minimally to heterogeneity. In a precision medicine context, the highest-importance covariate is the natural candidate for subgroup stratification: patients in the upper versus lower quantiles of that variable represent the population segment with the most divergent treatment responses. These importance scores should be interpreted as measures of predictive splitting activity, not as causal effect estimates for the covariates themselves.
